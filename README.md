# Claudi's Homebrew tap

## Available formulae

- [emacs-mac](https://github.com/jdtsmith/emacs-mac)
  - Split off the [original tap by railwaycat](https://github.com/railwaycat/homebrew-emacsmacport).
  - Inclues the [beautyfull Liquid Glass icons by jimeh](https://github.com/jimeh/emacs-liquid-glass-icons). 

## How do I install these formulae?

`brew install claudiiii/tap/<formula>`

Or `brew tap claudiiii/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "claudiiii/tap"
brew "<formula>"
```
